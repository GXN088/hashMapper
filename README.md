# hashMapper

Создайте метод с именем , который принимает вложенную HashMap под названием printNestedInventory в качестве входного параметра. В этом инвентаре:inventoryКлючи — это категории (например, "Electronics", "Furniture").Значения — это внутренние HashMap, где ключи — это названия продуктов, а значения — их цены.Ваш метод должен выводить инвентарь в следующем формате:Для каждой категории выведите Category: <name>Если категория имеет продукты, выведите каждый как  (с отступом в 2 пробела)Product: <name>, Price: <price>Если категория не имеет продуктов, выведите  (с отступом в 2 пробела)(No products)Пример вывода:

Category: Electronics
  Product: Laptop, Price: 1200
  Product: Smartphone, Price: 800
Category: Furniture
  Product: Chair, Price: 50
  Product: Table, Price: 150
Category: EmptyCategory
  (No products)



Примечание: HashMap не гарантирует порядок итерации, поэтому порядок категорий и продуктов в вашем выводе может отличаться от примера выше — это ожидаемо.Примечание: Вам потребуется импортировать  в дополнение к другим предоставленным импортам.java.util.Map
